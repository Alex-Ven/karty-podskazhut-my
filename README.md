# Cards Will Tell — Oblique Strategies Online (Educational Project)

**A digital version of the famous Oblique Strategies deck** by Brian Eno and Peter Schmidt.  
An interactive single-page website with creative prompt cards to overcome mental blocks.

**This is my educational project** — built from scratch to practice **HTML, CSS, and Vanilla JavaScript**.

Live Demo: [karty-podskazhut-my.vercel.app](https://karty-podskazhut-my.vercel.app)

---

## Technologies

| Category       | Technologies |
|----------------|--------------|
| **Language**   | HTML5, CSS3, Vanilla JavaScript |
| **Styling**    | CSS (BEM methodology), responsive layout, mobile-first |
| **Theme**      | Dark / Light mode (saved in LocalStorage) |
| **Fonts**      | Locally hosted (`./fonts/`) |
| **Optimization** | WebP, AVIF, lazy loading, favicon support |

---

## Features

- **Theme Toggle** (dark/light) with persistent user preference  
- **Prompt Card Display** (e.g., *"Use an old idea"*)  
- **Fully Responsive Design** (mobile, tablet, desktop)  
- **Fast Loading** thanks to modern image formats  
- **Semantic HTML** & accessibility (ARIA, alt texts)  
- **Cross-browser compatibility**
